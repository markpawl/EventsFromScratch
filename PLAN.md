# EventsFromScratch - Plan

This project was scaffolded from RNStarter features. Goals:

1. Implement an Expo-managed TypeScript app with the same UX as RNStarter.
2. Use `@expo/vector-icons`, simple stack navigation, and AsyncStorage for persistence.
3. Copy assets from RNStarter into `src/assets` for initial development.
4. Build components incrementally: PageContent, SetList, ArtistModal, VenueModal, Menu.

Next steps: implement core components and wire locater state and persistence.
